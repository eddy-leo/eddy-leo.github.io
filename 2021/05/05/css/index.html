@(Sample Introduction)[LCD|Android Studio]

在之前三个月VS项目中，我在团队中主要负责软件部分。

Learn Java and AS to write and generate APP.
In this project, the graphical interface was developed in Android Studio, using Java programming language.Its main function is to provide customers with a comprehensive game, allowing developers to check all aspects, including sensors, communications and servo motors.

The upper computer use a friendly provided LCD screen.这是LCD一些端口的示意图，如果还需要更详细的了解，可以去友善官方去查看。

Requirement of project

  • Two language modes - English mode and Chinese mode.
  • Two operation modes - User mode and administrator mode.
  • Fully automatic - The entire game is automated from start to finish.
  • User identification - By swiping the card.
  • Guide user operation - through appropriate voice and text prompts.
    大概最基础的就这几点,语言切换，模式，自动化，身份识别，语音提示，智能，哦对了还有通信，通信会在下一个part里说。

toast.show();//另一种方式

//用于多个组件监听时跳转页面
 private void setListener() {
        OnClick onClick = new OnClick();

    private class OnClick implements View.OnClickListener {

        @Override
        public void onClick(View v) {
            Intent intent = null;
            switch (v.getId()) {
                case R.id.btn_main_1:
                    intent = new Intent(SlideActivity.this, btn1.class);
                    break;
                case R.id.btn_main_6:
                    intent = new Intent(SlideActivity.this, btn6.class);
                    break;
                case R.id.btn_main_3:
                    intent = new Intent(SlideActivity.this, btn3.class);
                    break;
                case R.id.btn_main_4:
                    intent = new Intent(SlideActivity.this, btn4.class);
                    break;
            }
            startActivity(intent);
        }

Table

AS中常用的一些快捷键
| Item | Content |
| :——– | ——–:|
| Ctrl+G | 同时按下Ctrl+G快捷键弹出快速定位框，在框中输入行数点击OK即可快速切换到对应的行数 |
| Ctrl+/ | 选中某一行，同时按下Ctrl+/快捷键可以注释这一行 |
| Ctrl+F | 同时按下Ctrl+F快捷键，将在编辑页的顶部弹出类内快速搜索栏，可以快速定位类内的某个单词，支持联想查找 |
| Ctrl+J | 同时按下Ctrl和J快捷键，弹出快捷代码框 |
| Ctrl+Alt+L | 对当前类的所有代码进行格式化 |
| Ctrl+鼠标左键 | 此快捷键可以查看鼠标选中的类或方法 |
| Alt+回车Enter | 选中某个，同时按下Alt+回车Enter会有神奇的事情 |
